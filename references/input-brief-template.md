## Content constraints
- Must include:
- Must avoid:
- Must compare with:
- Must use real cases? (yes/no)
- Need boundaries / risks page? (yes/no)
- Need recommendation / next-step page? (yes/no)

## Production constraints
- Preferred format: (HTML/CSS / Markdown / PPTX outline / image-based)
- Need editable prototype? (yes/no)
- Need PDF export? (yes/no)
- Existing material to reuse:
- Brand assets available: (logo / avatar / template / colors / none)

## Decision check
Before starting page writing, confirm:
- Is the subject clear enough?
- Is the main storyline agreed?
- Are the most important proof points collected?
- Is the chosen output format the right one for expected iteration speed?
